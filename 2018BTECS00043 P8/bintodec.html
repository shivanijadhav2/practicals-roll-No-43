Binary to Decimal Conversion

Problem Statement :

Write a code in any programming language that converts Converts binary number to decimal number.

Algorithm:

1. The idea is to extract the digits of a given binary number starting from right most digit and keep a variable dec_value.
2. At the time of extracting digits from the binary number, multiply the digit with the proper base (Power of 2) and add it to the variable dec_value.
3. At the end, the variable dec_value will store the required decimal number.

Sample input & Output

 Sample input1:                            Sample input2:
  1111                                                     011
 Output1:                                  Output2:
  23                                                      3

Code:

  C++ Program : 
         
        #include <bits/stdc++.h> 
        using namespace std; 

        // Function to convert binary to decimal
        int binaryToDecimal(int n)
        {
            int num = n;
            int dec_value = 0;
        
            // Initializing base value to 1, i.e 2^0
            int base = 1;
        
            int temp = num;
            while (temp) 
            {
                int last_digit = temp % 10;
                temp = temp / 10;
        
                dec_value += last_digit * base;
        
                base = base * 2;
            }
        
            return dec_value;
        }
        
        // Driver program to test above function
        int main()
        {
            int num = 10101001;
        
            cout << binaryToDecimal(num) << endl;
        }

Time Complexity:

    The binary to decimal conversion algorithm is of complexity O(k), 
    i.e. the complexity of the loop. 
    where, k is a count of the number of digits in the binary representation of x,
    that is log2x, thus this can also be treated as a algorithm of complexity O(log x)

Space Complexity:

Space Complexity: O(1)
